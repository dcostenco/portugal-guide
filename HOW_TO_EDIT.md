# ИНСТРУКЦИЯ ПО УПРАВЛЕНИЮ САЙТОМ

Сергей, добро пожаловать в инструкцию!
Я сделал так, чтобы вы могли менять всё на вашем сайте самостоятельно, не зная программирования.

Вам нужно всего лишь изменять файл `content.json`, который находится в папке `src/data/`.

## Как изменить текст?
1. Откройте файл `src/data/content.json`
2. Вы увидите такие строки, например:
   `"whatsappUrl": "https://wa.me/351967202204"`
3. Если номер поменялся, просто аккуратно измените его внутри кавычек:
   `"whatsappUrl": "https://wa.me/351123456789"`
4. То же самое касается туров, цен (можете добавить поле `price`), описаний и отзывов.

## Как добавить новый тур?
В самом файле `content.json` есть блок `"tours": [...]`.
Просто скопируйте одну из готовых экскурсий и вставьте ниже (через запятую).
Например:
```json
{
  "id": "new-tour",
  "title": "Новое название",
  "duration": "3 часа",
  "route": "Точка А -> Точка Б",
  "description": "Описание экскурсии",
  "image": "/images/new.jpeg"
}
```

## Как обновить сайт?
Если ваш сайт подключен к Vercel через ваш аккаунт GitHub:
Как только вы сохраняете этот файл на сайте GitHub, система Vercel СРАЗУ ЖЕ сама начинает обновлять ваш сайт в интернете. Через 2 минуты изменения будут видны всем!
